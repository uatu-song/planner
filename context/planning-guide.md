## Teacher Context
**Position**: 8th Grade ELA Teacher  
**Courses**: General Ed 8th ELA, ELA 8/9 (Accelerated), plus one elective  
**School**: Plymouth Scholars Charter Academy, Plymouth, MI  
**Rankings**: #1 Charter Elementary & Middle School in Michigan  
**Demographics**: 51% Asian, 27% White, 9% African American, 9% Two or more races, 4% Hispanic  

## Core Challenges & Priorities

### Standards-Based Grading (SBG) Implementation
- **Philosophy**: Show mastery/highest achievement vs. averaging
- **System Limitations**: NHA gradebook lacks SBG features (no filtering by standards)
- **Resistance**: Teachers prefer traditional grading; parents want "more grades"
- **Strategy**: Create parallel tracking system, educate stakeholders continuously

### Workload Management
- **Issue**: Excessive duties beyond teaching overwhelming staff
- **Solution**: Leverage shared resources between courses, use common routines
- **Priority**: Maintain quality instruction while protecting personal sustainability

## Technology Integration Suite

### Primary Tool: Pear Deck (Daily Formative Assessment)
- **Every lesson** includes guided discussion questions
- **Real-time formative assessment** for SBG tracking
- **Student voice** captured from all learners, not just volunteers
- **Evidence collection** for standards mastery documentation

#### Pear Deck Implementation Strategy
1. **Pre-load** guided discussion questions into Pear Deck slides
2. **Launch** during Literature Seminar discussions
3. **Monitor** responses in real-time to adjust instruction
4. **Export** responses for SBG evidence folders
5. **Review** patterns to inform next day's instruction

### Supplemental Tool: Blooket (Engagement & Review)
- **Purpose**: Vocabulary review, comprehension checks, unit reviews
- **Frequency**: 1-2 times per week, especially before assessments
- **Best Modes** for ELA:
  - **Tower Defense**: Vocabulary practice (Unit 2: Chew On This)
  - **Café**: Character analysis questions (Units 3, 5, 6)
  - **Gold Quest**: Plot sequencing (Unit 1: Short Stories)
  - **Factory**: Theme identification (all units)

#### Blooket Strategic Implementation
- **Monday**: Preview vocabulary for the week
- **Thursday**: Review comprehension before Friday assessments
- **Pre-Assessment**: Unit review games
- **Differentiation**: Gen Ed gets more time/hints; 8/9 gets harder questions

### Supplemental Tool: NoRedInk (Grammar & Writing)
- **Purpose**: Targeted grammar practice aligned to writing units
- **Frequency**: 2-3 assignments per week (10-15 min each)
- **SBG Alignment**: Direct connection to Language standards (MT 5)

#### NoRedInk Unit Alignment
- **Unit 1**: Sentence structure, comma usage (for constructed responses)
- **Unit 2**: Conjunctions, transitions (for argumentative writing)
- **Unit 3**: Dialogue punctuation, verb tenses (for historical narrative)
- **Unit 4**: Semicolons, complex sentences (for research writing)
- **Unit 5**: Active/passive voice, tone (for satire)
- **Unit 6**: Shakespearean language structures, poetic devices

### Creative Tool: Canva (Visual Projects & Presentations)
- **Purpose**: Professional visual products for culminating tasks
- **Access**: Education accounts (free for teachers/students)
- **SBG Alignment**: Demonstrates synthesis and higher-order thinking

#### Canva Project Applications by Unit

#### Canva Templates for Efficiency
Create unit-specific templates:

- **Argument Map** template (Unit 2)

#### Differentiation with Canva
**Gen Ed 8th Grade**:
- Provide more structured templates
- Offer specific element requirements

**ELA 8/9**:
- Open-ended creative freedom
- Multi-page presentations

- Professional publication standards

### Tech Stack Workflow Integration

#### Weekly Rhythm
**Monday**: 
- Introduce unit vocabulary via Blooket
- Assign NoRedInk grammar topic for the week

**Tuesday-Wednesday**:
- Daily Pear Deck guided discussions
- NoRedInk practice (homework/station rotation)
- Google Docs for writing workshop

**Thursday**:
- Blooket review game
- Complete NoRedInk assignment
- Canva work time for projects (if applicable)

**Friday**:
- Pear Deck synthesis/assessment
- Project presentations (when applicable)
- Export all week's data for SBG tracking

#### Major Project Checkpoints
**Week 1 of Project**: Introduce Canva template, brainstorm
**Week 2 of Project**: Peer feedback via Google Forms
**Week 3 of Project**: Canva creation time
**Week 4 of Project**: Gallery walk/presentations

### Data Collection for SBG

#### Weekly Evidence Portfolio
1. **Pear Deck Exports**: Reading comprehension (MT 1), Analysis (MT 2)
2. **Blooket Reports**: Vocabulary mastery, comprehension speed
3. **NoRedInk Data**: Grammar/language progress (MT 5)
4. **Canva Projects**: Synthesis and creativity (MT 2, 3)
5. **Google Docs**: Writing process and revision (MT 3)
6. **Combined View**: Holistic picture of student progress per standard

#### Parent Communication
- Share Blooket game codes for home practice
- NoRedInk parent reports show specific skill progress
- Pear Deck Takeaways sent home weekly
- Canva projects shared via Google Classroom showcase
- All tools provide concrete evidence for conferences

### Time-Saving Setup Tips
1. **Batch Creation Sundays**: Build week's Pear Decks, Blookets, and NoRedInk assignments
2. **Template Everything**: Save question sets and Canva templates for reuse
3. **Student Helpers**: Have 8/9 students create Blooket sets and Canva examples for Gen Ed
4. **Auto-Assignment**: Schedule NoRedInk assignments in advance
5. **Cross-Platform**: Use same vocabulary lists across all tools
6. **Canva Brand Kit**: Create class colors/fonts for consistency
